# Minecraft-Closet-Mods
Holy Closets

### Commands

| Command | What it does |
|---|---|
| `/pvp aim` | Toggle aim assist |
| `/pvp crit` | Toggle criticals |
| `/pvp crit packet` | Set crit mode to packet |
| `/pvp crit jump` | Set crit mode to jump |
| `/pvp shield` | Toggle shield disabler |
| `/pvp hitselect` | Toggle hit select |
| `/pvp friend add <name>` | Add friend |
| `/pvp friend remove <name>` | Remove friend |
| `/pvp friend list` | List friends |
| `/pvp friend clear` | Clear all friends |
| `/pvp status` | Show status of all modules |
